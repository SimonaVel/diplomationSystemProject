<h2>Introduction</h2>
This is a Spring boot REST-full web application automating the process of diplomation for both soon-to-graduate university students and tutors in a given university department.

<h3>Context</h3>
This project was being developed during the autumn and spring semesters of 2024/2025 for 2 courses - Java Web Services and Realisation of software projects. 
A detailed description of the existing functionalities + a Kanban board are available -> documentation : CSCB869_Kostadinova_Graduation_System.pdf<br>
